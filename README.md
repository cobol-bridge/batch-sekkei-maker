# バッチ設計書メーカー

COBOLバッチ処理のソースコードを読み込んで、業務設計書（Excel/Word）を自動生成するツールです。

## 特徴

- 完全オフライン動作（社外秘コードを外部送信しない）
- COBOLソースを選択するだけで設計書が自動生成
- Excel / Word 両形式で出力
- Windowsデスクトップアプリ（.exe）

## 出力される設計書

| シート | 内容 |
|--------|------|
| 表紙 | ファイル名・生成日時・基本情報 |
| I/O定義 | インプット/アウトプットのファイル定義一覧 |
| 処理フロー | PERFORMの呼び出し階層 |
| 例外処理 | INVALID KEY / AT END 等の一覧 |

## 動作環境

- Windows 10 / 11
- インターネット接続不要

## 開発環境のセットアップ

```bash
# 仮想環境を作成
python -m venv venv

# 仮想環境を有効化
venv\Scripts\activate

# ライブラリをインストール
pip install -r requirements.txt
```
