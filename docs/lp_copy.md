## 解決策セクション

### 見出し
```
バッチ設計書メーカーなら、数秒で解決します
```

### 説明
```
COBOLソースファイルを選択して「生成する」を押すだけ。
SELECT句・FD句・PERFORM呼び出し・例外処理を自動解析し、
業務設計書のたたき台を即座に生成します。
```

## 機能セクション

### 見出し
```
生成される設計書
```

### 機能カード

**表紙**
```
対象ファイル名・生成日時・バージョンを自動記入
```

**I/O定義**
```
SELECT句からファイル名・物理名・編成・アクセスモード・
レコードキーを一覧化
FD句からレコード長を補完
```

**処理フロー**
```
PERFORM呼び出し階層を可視化
THRU・UNTIL・VARYINGにも対応
```

**例外処理**
```
INVALID KEY / AT END / ON SIZE ERROR 等を一覧化
発生箇所（段落名）付きで出力
```

## FAQ

**Q: どんなCOBOLに対応していますか？**
A: IBM Enterprise COBOL（固定形式）に対応しています。文字コードはShift-JIS / UTF-8 を選択できます。

**Q: 生成した設計書をそのまま納品できますか？**
A: 設計書はあくまでたたき台です。内容は必ず目視で確認・加筆してからご利用ください。
